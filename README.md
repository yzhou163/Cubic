# Cubic
Cubic is a creative distortion based audio plugin built with the JUCE framework.  It features an overdrive distortion module, sample rate reduction module, and a ring modulation.

Requirements
- JUCE (8.0.6) installed
- Xcode (macOS) or Visual Studio (Windows)
